# لوحة تذكارية - Memorial Plaque Website

موقع مستقل لبيع اللوحات التذكارية الفاخرة

## 📁 هيكل المشروع

```
mom/
├── index.html          # الصفحة الرئيسية
├── images/            # مجلد الصور (أضف صور المنتجات هنا)
└── README.md          # هذا الملف
```

## ✨ المميزات

### التصميم
- تصميم أنيق ومحترم مناسب للوحات التذكارية
- ألوان هادئة (أزرق، رمادي، أبيض)
- تجاوب كامل مع جميع الأجهزة
- رسوم متحركة سلسة

### نظام الطلبات
- نموذج طلب مخصص للوحات التذكارية
- تكامل مع Firebase Firestore لحفظ الطلبات
- إرسال تلقائي للواتساب
- قاعدة بيانات كاملة للولايات والبلديات الجزائرية

### خيارات التخصيص
- الاسم على اللوحة
- التاريخ (اختياري)
- رسالة أو عبارة (اختياري)
- اختيار التصميم (كلاسيكي، عصري، أنيق)
- نوع التوصيل (منزل أو مكتب)

## 📸 إضافة الصور

1. أضف صور اللوحات التذكارية في مجلد `images/`
2. قم بتحديث الصور في ملف `index.html`:
   - ابحث عن `<div class="w-full h-64 bg-gradient-to-br..."`
   - استبدل بـ: `<img src="images/اسم_الصورة.jpg" alt="...">`

## 🚀 الاستخدام

1. افتح `index.html` في المتصفح
2. أو ارفع المجلد `mom` على الاستضافة
3. الموقع جاهز للعمل مباشرة!
